# Концепции — ComminusmPlugin

Эта папка содержит общий документ по архитектуре и решениям.

## Содержимое

- [[concepts/architecture|Архитектура]] — overview проекта и технологии
- [[concepts/decisions/|Папка ADR]] — записи архитектурных решений

## Описание концепций

Концепции объясняют "что" и "почему" проекта. Это общий документ для понимания:

- Общая архитектура системы
- Принятые решения (ADRs)
- Фундаментальные принципы

## Как использовать концепции

1. Начните с [[concepts/architecture|Архитектуры]] — поймите общую структуру
2. Изучите [[concepts/decisions/|ADR]] — поймите принятые решения
3. Перейдите к [[how-to|Как сделать]] — примените знания на практике
4. Следуйте [[guidelines|Правилам]] — соблюдайте конвенции
