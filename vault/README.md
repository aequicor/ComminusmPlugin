# Документация проекта ComminusmPlugin

Эта папка содержит всю документацию проекта ComminusmPlugin, организованную по фреймворку [Diátaxis](https://diataxis.fr/).

**Документация — единственный источник истины.** Если что-то не документировано, это считается отсутствующим знанием.

## Жанры (Genres)

### reference/ — Справочник

Факты и конкретная информация. Содержит:
- API документацию
- Конфигурационные файлы
- Примеры кода

### reference/dev-setup

Инструкции по настройке среды разработки.

## Как использовать

1. Начните с `_INDEX.md` — главного индекса
2. Перейдите в нужный жанр
3. Следуйте вики-ссылкам между документами
4. Обновляйте документацию вместе с кодом
