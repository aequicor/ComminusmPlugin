# Идеология проекта

## Миссия

Создать сатирический Minecraft-плагин, который геймифицирует идеи коммунизма для развлекательных целей, неся просветительскую нагрузку через игровую механику.

## Ценности

1. **Коллективизм** — всё ресурсы общие, распределение по потребностям
2. **Равенство** — все игроки равны перед лицом партии (сервера)
3. **Братство** — взаимопомощь и поддержка товарищей
4. **Сатира** — критика экстремальных идей через юмор и преувеличение

## Целевая аудитория

- Игроки PaperSpigot серверов (Minecraft 1.21.11+)
- Любители сатирического контента
- Разработчики, изучающие создание плагинов

## Техническая конфигурация

- **Paper** 1.21.11 (последняя стабильная версия)
- **ViaVersion** 5.9.0 (для совместимости с клиентами 26.x)
- **Java** 25
- **Kotlin** 2.3.20-RC3

## Идеологическое ядро

- Общая казна (весь опыт и валюта)
- Равное распределение ресурсов
- Запрет на частную собственность
- Трудовая повинность
- Продовольственные карточки

## Дизайн-принципы

- **Увесистость** — преувеличение ироничных идей
- **Шуточность** — юмор вместо жесткости
- **Просветительность** — сквозь смех — разум

## Правила взаимодействия

- Все решения документируются в `vault/`
- Используется фреймворк Diátaxis для организации знаний
- Код пишется на Kotlin с использованием PaperSpigot API
