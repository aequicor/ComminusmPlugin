# CommunistRules ☭

Первый в мире плагин для Paper 1.12.2, который устанавливает истинный режим цифрового коммунизма на вашем сервере Minecraft. Забудьте о частной собственности — теперь всё принадлежит партии (серверу).

## Основные функции

*   **Общая казна:** Весь опыт и валюта игроков переходят в общий фонд.
*   **Равное распределение:** Ресурсы из сундуков автоматически перераспределяются между нуждающимися товарищами.
*   **Запрет на частную собственность:** Попытка заприватить территорию карается отправкой в ГУЛАГ (специальный мир/тюрьму).
*   **Продовольственные карточки:** Еда выдается строго по расписанию через системные сообщения.
*   **Трудовая повинность:** Игроки обязаны добывать определенное количество ресурсов в день для общего блага.

## Установка

1. Скачайте `.jar` файл плагина.
2. Поместите его в папку `plugins` вашего сервера Paper 1.12.2.
3. Перезапустите сервер.
4. Наслаждайтесь равенством и братством!

## Команды

*   `/comrade report <player>` — донести на товарища за хранение алмазов.
*   `/party supply` — получить свою норму хлеба.
*   `/gulag list` — список врагов народа на текущий момент.

## Требования

*   **Платформа:** PaperSpigot
*   **Версия:** 1.12.2 (и выше)
*   **Дух коллективизма:** Обязателен.
